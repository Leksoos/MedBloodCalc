Системные требования
Минимальные требования:
•	ОС: iOS 13.0 или Android 9.0+;
•	ОЗУ: 2 ГБ;
•	Свободное место: 200 МБ;
•	Интернет: не требуется для основных функций (только для обновления приложения).
Рекомендуемые требования:
•	ОС: iOS 14.0+ или Android 11.0+;
•	ОЗУ: 4 ГБ;
•	Процессор: современный (выпуска 2019 г. и позже).
